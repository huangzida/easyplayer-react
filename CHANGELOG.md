## [0.1.0] - 2026-04-06

### Features

- Initial release of easyplayer-react
- React wrapper for EasyPlayerPro with typed props, events, and instance methods
- Support for HLS, FLV, MP4, WebM streaming formats
- Built-in VOD and Live mode presets
- Full TypeScript support
- React 17+ compatible
- Configurable playground for testing
